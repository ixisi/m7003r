# m7003r
M0073R (Auto Muter) 

is a simple Lua script that automatically mutes or unmutes a selected audio source when you switch to a specific scene. 

Perfect for ensuring your desktop audio or secondary mic is silenced on your "Be Right Back" or "Alerts" scenes



How To Use:

Once you loaded the script it will display the current scene available in the scene collection

If you inserted an audible source that you want to mute/umute it will show it in the list below the specific scene you selected.

**( undefined state )** Will only show up if you have not interacted with check box
**( mute state )** Will only show up if you have checked the box
**( unmute state)** Will only show up if you have not checked the box


Click on the **'+'** plus button to add a new audible source
[Example Three](assets/m7003r_example3.png)



**[ Audible ]** This list will only show the audible sources that you can select

**[ Scene ]** This list will show all the scenes on the scene collection

**[ Confirm ]**  Click this button to save the selection

**[ Cancel ]** Click this button to cancel



**Notice: To delete the insertion, you have to double click the check box**
